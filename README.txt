# Auto Form Filler - Project Setup Guide

This is an LLM-powered Auto Form Filler. It extracts your CV data using GPT and automatically fills a Google Form.

---

## 🚀 Prerequisites

1. Python 3.8 or higher
2. Google Chrome browser
3. ChromeDriver (provided in the project folder, must match your Chrome version)
4. OpenAI API key (get it from https://platform.openai.com/account/api-keys)
